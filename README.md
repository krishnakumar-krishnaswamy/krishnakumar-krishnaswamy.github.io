# Portfolio Website

Learning, experimenting, and building pilot initiatives in modern digital transformation and enterprise delivery.

## Tech Stack

### HTML
Provides the structural foundation of the portfolio including layout and content organization.

### CSS
Defines visual styling, spacing, typography, and layout behavior.

### JavaScript (Vanilla)
Uses plain JavaScript without frameworks to handle navigation, dynamic rendering, and user interaction.
